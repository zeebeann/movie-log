# Movie Ticket Log

A simple movie-logging interface that lets users record the films they watch and view them as custom **CSS-styled movie tickets**.
Built and designed by **Zainab Ilyas** as part of Sheridan College’s *System Design: Databases and Dynamic Data* course.

---

## Features

* Log a film by entering:

  * **Movie Title** (with autocomplete)
  * **Rating**
  * **Date Watched**
* Automatically fetches the **movie poster**
* Stores all entries in **MongoDB**
* Displays logged films as **CSS movie tickets**

---

## Data

* **Movie Data & Posters**: [TMDB API](https://www.themoviedb.org/documentation/api) (The Movie Database™)
